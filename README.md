# drinkmix-data-warehouse
## Overview

This project implements a Data Warehouse for a beverage retail chain using Oracle SQL.

## Architecture
Star Schema consisting of:

- Fact Sales
- Customer Dimension
- Product Dimension
- Time Dimension
- Branch Dimension

## ETL Features

- Customer deduplication across branches
- Address standardization
- Product harmonization
- Category normalization
- Fact table loading

## Results

- 141,753 integrated customers
- 212,622 sales records
- Turnover and sales volume reporting

## Technologies

- Oracle SQL
- Data Warehousing
- ETL
- Dimensional Modeling
